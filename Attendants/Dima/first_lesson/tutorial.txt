1. Create package.json file: npm init -y
2. npm i -D typescript ts-node @types/node @tsconfig/node22
	- typescript - package that allows compilation from typescript to JS
	- ts-node allow tunning code without compilation to TS, not allowed in the production enviroment.
	- @types/node - manual implementation of the type`s support
3. Creating ts.config file: code ./tsconfig.json
	- here we can change the JS versions is used
4. Configurating eslint: npm init @eslint/config
